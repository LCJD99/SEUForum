# SEUForum backend

## Develop with Docker

### Requirements

- Docker
- VSCode

#### VSCode Extensions
- wsl
- docker
- mongo

#### Start demo
1. 克隆到本地 `wsl` 中
``` bash
git clone -b dev1.0 https://github.com/LCJD99/SEUForum.git
```
2. 在根目录构建docker镜像
``` bash
docker compose up -d
```
